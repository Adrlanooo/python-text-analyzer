# Python Text Analyzer

A simple Python program that analyzes text and provides useful insights.

## Features

* Counts total words
* Counts characters and letters
* Counts vowels and consonants
* Finds the longest word

## How to Run

1. Make sure Python is installed
2. Run:

python text_analyzer.py

## Example Output

Total Words: 7
Letters: 45
Vowels: 22
Consonants: 23
Longest Word: eagleeeeeee
